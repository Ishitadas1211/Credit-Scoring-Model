# Credit Scoring Model

## Project Description
This project performs data preprocessing and analysis on the German Credit dataset using Python.

## Features
- Load the dataset
- Handle missing values
- Encode categorical columns
- Save the cleaned dataset
- Display summary statistics
- Generate visualizations

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Project Files
- Credit-Scoring.py
- german_credit_data.csv
- requirements.txt
- README.md

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the project:

```bash
python Credit-Scoring.py
```

## Output
- Cleaned dataset
- Dataset statistics
- Age distribution graph
